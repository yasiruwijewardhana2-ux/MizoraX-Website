# MizoraX
A futuristic anime/movie streaming platform UI with a unique Japanese cyber style.

Features:
- Vanilla HTML/JS/CSS
- TailwindCSS (via CDN)
- Glassmorphism & Neon Design
- LocalStorage based Admin System
- Responsive design

## Setup
No build step required. Just open `index.html` in any web browser.
Admin login: `admin` / `admin`
